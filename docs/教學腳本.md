# 使用教學腳本(錄影用)

用手機螢幕錄影照著操作 + 念旁白即可,每支約 60–90 秒。
(App 內也有「❓ 使用教學」圖文版:玩家在大廳「你的資料」、團主在「設定」裡。)

---

## 🏸 臨打人教學(約 75 秒)

| # | 畫面操作 | 旁白 |
|---|---------|------|
| 1 | 開 App,點 Google / LINE 登入 | 第一次用,先用 Google 或 LINE 登入,系統會幫你建好資料。 |
| 2 | 首頁點一個進行中的開團(或掃 QR)→ 輸入密碼 | 從首頁點要去的球團,輸入團主給的密碼就進場。 |
| 3 | 點球場空位「＋」上場 | 點球場上的空位就上場,湊滿四個人就開始算時間。 |
| 4 | 指一下右上鈴鐺 | 輪到你上場會跳通知,鈴鐺裡看得到這場的紀錄。 |
| 5 | 上方「＋帶家人」填名字 / 頭像 / 程度 | 沒帶手機的家人?點「帶家人」幫他加入,團主核准後就能一起排。 |
| 6 | 四人在場 → 點「投票結束這場」 | 打完想換,場上點「投票結束」,三票就自動換下一組。 |
| 7 | 右上「設定」改名字 / 頭像 | 想改名字、換頭像,右上設定就可以。 |

---

## 🧑‍🏫 團主教學(約 90 秒)

| # | 畫面操作 | 旁白 |
|---|---------|------|
| 1 | 首頁填開團名稱 / 縣市 / 密碼 / 時間 / 場數 → 開團 | 開新團:填名稱、地點、進場密碼、時間、幾片場地,按開團。 |
| 2 | 場次頁上方 QR / 複製連結 / 海報模式 | 把這個 QR 或連結給大家,海報模式可以放在門口。 |
| 3 | 本場人員點一個人 → 改程度 / 💰臨打費 | 點任何一個人,可以改程度、標記收了臨打費、或斷開。 |
| 4 | 球場「手動加人」或「現場排點板」 | 幫人排上場用手動加人;沒帶手機的人用現場排點板代排。 |
| 5 | 點「結束這場」→(指)「還原」 | 一場打完點結束換下一組;按錯十分鐘內可以還原。 |
| 6 | 待核准的家人 → 點核准 | 有人帶家人會出現待核准,點一下核准就生效。 |
| 7 | 點場次標題改名 | 想改開團名字?直接點標題就能改。 |
| 8 | 右上「總結」 | 散場點總結,看每個人打了幾場。 |

## 錄製小撇步
- iPhone:控制中心「螢幕錄製」;Android:下拉快捷「螢幕錄影」。
- 先照腳本走一遍再正式錄;旁白可錄完畫面再配音。
- 想要 30 秒短版:臨打人留 #2 #3 #6、團主留 #1 #2 #4 #5 即可。
